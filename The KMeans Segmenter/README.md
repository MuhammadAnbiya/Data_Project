# The K-Means Segmenter

Studi kasus ini akan membawa Anda memahami konsep dasar Unsupervised Learning menggunakan **K-Means Clustering**.

## Tujuan Pembelajaran
1. Memahami cara kerja algoritma pengelompokan tanpa label kelas.
2. Memahami logika inisialisasi *Centroid* (titik pusat).
3. Mengimplementasikan perulangan pembaruan *Centroid* hingga stabil (Converge).

Buka `kmeans_clustering.py` untuk mulai memecahkan masalah ini!
