# The KNN Matchmaker

Studi kasus ini dirancang untuk melatih pemahaman Anda tentang algoritma K-Nearest Neighbors (KNN) dari awal (scratch) tanpa bantuan library Machine Learning seperti Scikit-Learn.

## Tujuan Pembelajaran
1. Memahami konsep kedekatan/kemiripan data (Euclidean Distance).
2. Memahami bagaimana algoritma KNN mencari tetangga terdekat.
3. Memahami proses "Voting" mayoritas untuk klasifikasi.

Silakan buka `knn_classifier.py` untuk mulai mengerjakan!
